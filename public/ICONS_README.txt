Push Notification Icons Required
=================================

For browser push notifications to display properly, you need to add the following icon files to this /public directory:

1. motorated-icon-192.png - 192x192 pixel PNG icon
2. motorated-icon-512.png - 512x512 pixel PNG icon
3. motorated-badge-72.png - 72x72 pixel badge icon (monochrome)

These icons should feature the MotoRate logo or branding.

If icons are missing, push notifications will still work but may not display an icon.

You can create these using:
- Online tools like https://www.favicon-generator.org/
- Design software like Figma, Photoshop, or Canva
- The MotoRate logo converted to different sizes

Temporary workaround:
- The app will function without icons
- Browser may show a default notification icon
- Add proper icons before production deployment
